# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2026-01-07
### Fixed
- Optimized UI updates to prevent freezing on high-performance devices during benchmarks.

## [1.0.1] - 2026-01-04
### Fixed
- Corregido el nombre del dispositivo "Pixel 7 Pro" hardcodeado en los resultados.

## [1.0.0] - 2026-01-02
### Added
- Lanzamiento inicial de **NPU Check**.
- Soporte para modelos: TinyStories, TinyLlama, y Gemma 2 2B.
- Métricas en tiempo real: Tokens por segundo (TPS), uso de RAM.
- Interfaz temática Cyberpunk.
- Sistema de descargas resumibles y gestión de caché.
